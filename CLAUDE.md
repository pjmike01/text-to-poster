# Text to Poster Generator

将文字内容生成 3:4 竖版杂志风 HTML5 横向滑动海报，并截图产出多张图片。

## 触发方式

直接描述你的需求，例如：
- "将这段文字生成海报"
- "把这篇内容做成杂志风海报"
- "生成海报并截图"

## 工作流程

1. **分析内容** - 理解用户提供的文字内容结构
2. **生成 HTML** - 根据设计规范生成海报 HTML 文件
3. **截图输出** - 使用 Playwright 将 HTML 的每一页截图保存为图片

## 海报设计规范

生成的海报必须遵循以下设计规范：

- **比例**：3:4 竖版，横向滑动翻页
- **风格**：瑞士主义非对称网格排版，纸媒特种纸质感
- **滤镜**：全局叠加低透明度噪点滤镜（`opacity: 0.08`, `multiply`）
- **配色**：
  - 基底色 `#F1F0EC`
  - 强调色 `#FF7D00`（橙色高亮）
  - 深色卡片 `#121212` 底色、白色文字
  - 禁止使用纯白
- **字体**：
  - `Oswald`（英文标题）
  - `Noto Serif SC`（中文标题粗体）
  - `Noto Sans SC`（正文）
- **布局**：横向滑动翻页，卡片锁定比例 `flex-shrink: 0`，一页一个核心模块，浅色深色卡片交替
- **样式**：浅灰色圆角卡片区块包裹，关键词/关键句橙色高亮，标题巨大醒目、正文清晰易读
- **内容**：完整展示全部文字，不删减、不概括、不简化，保持原文结构与段落，分页面清晰呈现

## 技术实现

- 纯 HTML + Tailwind CSS
- 单文件包含全部样式
- 使用 Google Fonts CDN 加载字体
- 无需外部依赖，可直接打开预览

## 输出

- **HTML 文件**: `output/poster-<timestamp>.html`
- **图片文件**: `output/poster-<timestamp>-page-1.png`, `poster-<timestamp>-page-2.png` 等

## 截图依赖

截图功能需要 Node.js 和 Playwright：

```bash
# 安装依赖
npm install

# 安装 Playwright 浏览器
npx playwright install chromium
```
